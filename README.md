# Vanja Cekić | Portfolio

**Live Website:** [https://imakamai.github.io/My-Portfolio/](https://imakamai.github.io/My-Portfolio/)

A minimal, brutalist-inspired personal portfolio website built to showcase my expertise in full-stack and backend development, emphasizing clean architecture and efficiency.

## 🚀 Overview

This project highlights my academic background, professional internships (such as my time at Sky Soft d.o.o. and Nignite), technical skills, and key projects like the **Dog Show Management System** and a **To-Do Web Application**.

## ✨ Features

- **Zero-Framework Architecture:** Built entirely utilizing pure HTML5, vanilla CSS3, and vanilla JavaScript.
- **Brutalist Minimalism:** A striking, ultra-readable aesthetic defined by sharp typography, off-white backgrounds, deep dark text, and vibrant blue accents.
- **Fully Responsive:** Smooth scaling and carefully designed media queries to support desktop, tablet, and mobile layouts dynamically.

## 🛠️ Setup & Usage

Since this is a completely static site with no backend dependencies or build steps, deploying or viewing it locally is extremely simple:

1. Clone or download this repository.
2. Locate the `index.html` file in the root directory.
3. Open it in your favorite web browser!
